# Mercury

Mercury is a free and open sourced Roblox UI library.

Code:
```lua
-- Last Update: 1/17
local Mercury = loadstring(game:HttpGet("https://raw.githubusercontent.com/deeeity/mercury-lib/master/src.lua"))()
```

## Features:
- Customizable (Themes)
- Browser-Like navigation
- Buttons
- Sliders
- Dropdowns
- Inputs
- Color Picker

... more to be added.
